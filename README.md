# FHEVM Vesting

A Confidential Vesting contract using Fully Homomorphic Encryption (FHE) using Zama fhEVM.

## Overview

An example of a vesting contract that do not reveal released nor releasable amounts.

## Quick start

```bash
npm i
npm t
```
